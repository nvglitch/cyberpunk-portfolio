# Cyberpunk Portfolio

一个具有赛博朋克风格的个人作品集网站，展示设计师的专业技能和项目作品。

## 📁 项目结构

```
cyberpunk-portfolio/
├── index.html          # 主页面
├── css/
│   └── styles.css      # 自定义样式
├── js/
│   └── main.js         # JavaScript功能
├── unnamed.jpg         # 个人头像
└── README.md           # 项目说明
```

## 🎨 设计特点

### 视觉风格
- **赛博朋克美学**：霓虹绿色、蓝色和红色的配色方案
- **故障艺术效果**：标题文字的故障动画
- **科技感界面**：HUD风格的个人信息面板
- **动态背景**：网格背景、扫描线效果和二进制动画
- **自定义光标**：十字准星光标，增强科技感

### 交互体验
- **流畅动画**：使用GSAP实现的滚动动画和过渡效果
- **响应式设计**：适配不同屏幕尺寸
- **导航系统**：粘性控制台导航
- **加载动画**：系统初始化风格的加载界面
- **悬停效果**：作品卡片和按钮的交互反馈

## 🛠 技术栈

- **前端框架**：纯HTML5 + CSS3 + JavaScript
- **CSS框架**：Tailwind CSS v3
- **动画库**：GSAP (GreenSock Animation Platform)
- **字体**：
  - Orbitron (标题)
  - Roboto Mono (正文)
- **图标**：自定义CSS实现

## 🚀 快速开始

### 本地预览

1. 克隆或下载项目到本地
2. 直接在浏览器中打开 `index.html` 文件

### 开发环境

项目使用CDN加载依赖，无需安装额外包：
- Tailwind CSS 通过CDN引入
- GSAP 通过CDN引入

   - 页面标题：`index.html` 第7行

6. **自定义光标**：十字准星风格，增强科技感
